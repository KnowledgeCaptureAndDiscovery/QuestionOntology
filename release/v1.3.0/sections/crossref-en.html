Question Variablec back to ToC or Class ToC

IRI: https://w3id.org/sqo#QuestionVariable

Class that represents a variable used on a Question. This variable name must be present on the template or pattern of the question that uses it. Can have contraints in the form of a query (hasOptionsQuery property) or a list of possible options (hasOptions property).
has sub-classes
Bounding Box Question Variable c, Time Interval Question Variable c, User Input Question Variable c
is in domain of
Has options op, Has options query dp, Has representation dp, Has variable name dp, has maximum cardinality dp, has minimum cardinality dp
is in range of
Has maximum latitude op, Has maximum longitude op, Has minimum latitude op, Has minimum longitude op, Has question variable op
